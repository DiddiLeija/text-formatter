# text_formatter

Pretty formatter for Python strings.

## Installation

The **`text_formatter`** package can be installed using **[`pip`](http://pip.pypa.io)**:

```
pip install text_formatter
```

\(To upgrade, use `pip install --upgrade text_formatter`\).
